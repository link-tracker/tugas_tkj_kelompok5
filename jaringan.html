ADMINISTRASI SERVER

Membangun dan mengelola infrastruktur server jaringan skala industri.

Apa yang Dipelajari?

Di bagian ini, siswa fokus pada pengelolaan sistem operasi server seperti Linux (Debian/Ubuntu) dan Windows Server untuk melayani kebutuhan jaringan lokal maupun publik.

Materi Utama:

  • Konfigurasi Layanan: DHCP, DNS, FTP, dan Web Server (Apache/Nginx).
  • Manajemen User: Mengatur hak akses dan keamanan data di server.
  • Remote Access: Mengontrol server jarak jauh menggunakan SSH.
